IndexedDB を活用したメモアプリです。

## 制作背景

「[**N予備校 動く Web ページコンテスト 2021 夏**](https://progedu.github.io/webappcontest/2021/summer/result/index.html)」に応募する作品として制作しました。  
当時興味があったクライアントサイド Web API を詰め込んで作った覚えがあります。  
N・S高等学校部門に応募して健闘賞をいただきました。

応募当時の作品紹介：  
IndexedDB API、Notifications API、Web Workers API、Web Components などのおもしろそうな機能を使ってみたくて作ったメモアプリです。`contenteditable` 属性を利用した WYSIWYG エディタの実装が大変でした。

## 主な機能

- メモの作成、編集、削除、検索
- IndexedDB API を用いたメモの永続化
- ストレージの使用量の可視化と管理機能
- リッチテキストに対応
- 画像や動画などのファイル添付に対応
- `contenteditable` 属性や Selection API などを用いた WYSIWYG エディタ
