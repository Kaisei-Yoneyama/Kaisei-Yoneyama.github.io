日々の出来事を画像や動画と一緒に記録できる日記アプリです。

## 制作背景

プログラミングを始めて最初に作った作品です。

当時、日記をつけるのがマイブームで写真や動画も一緒に保存できる日記帳が欲しいと思い制作した覚えがあります。

「[**N予備校 動く Web ページコンテスト 2020 夏**](https://progedu.github.io/webappcontest/2020/summer/result/index.html)」N・S高等学校部門に応募して健闘賞をいただきました。

## 主な機能

- 日記の作成、編集、削除、検索
- ローカルストレージを用いた日記の永続化
- File API や `contenteditable` 属性を使用した画像・動画ファイルの読み込みと埋め込み
