作成した Web アプリケーションなどの作品を紹介・共有できるポートフォリオプラットフォームです。  
初めて作ったクライアントサーバシステムの Web アプリケーションです。

## 制作背景

「[**N予備校 Web アプリケーションコンテスト 2021 冬**](https://progedu.github.io/webappcontest/2021/winter/result/index.html)」に応募する作品として制作しました。  
N・S高等学校部門に応募して優秀賞をいただきました

応募当時の作品紹介：  
作ったアプリを紹介できるサービスです。特にアプリのサムネイルを表示する機能を頑張って作りました。今後も Markdown を使えるようにするなど、いろいろな機能を追加していきたいと思います。

## 主な機能

- GitHub アカウントでのログイン
- 作品の投稿/編集/削除
- 作品のサムネイル画像のアップロード
- 全ての作品の一覧/詳細表示
- ユーザーごとの作品の一覧/詳細表示
- 作品へのコメント (Markdown 対応)
- 管理者機能
